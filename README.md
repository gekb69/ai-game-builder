```bash

make setup

# Run system
make run

# View features
make list-features

# Simulate emergency
make emergency-test
```

- `POST /api/task` - Submit task
- `GET /api/status` - System status
- `POST /api/emergency/simulate` - Simulate emergency
- `POST /api/consciousness/benchmark` - Run benchmark
